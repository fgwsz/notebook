主窗口(默认)
{
alt         进入标签选择模式 
            tab     按数次table键实现标签的切换，按回车键使用（类似于鼠标左键双击）选中的标签
			
↑           (在聊天小窗口打开的情况下)切换到上一个联系人，并进入对应的聊天框的输入框
↓           (在聊天小窗口打开的情况下)切换到下一个联系人，并进入对应的聊天框的输入框
esc         主窗口最小化
alt+ctrl+w  取消主窗口最小化，弹出主窗口
alt+m 进入聊天框的消息列表
{
	↑     切换到上一条消息
	↓     切换到下一条消息
	enter 如果当前选中的一条消息是文件，会打开该文件
          如果当前选中的一条消息是网页，会打开该网页	
		  
	alt+s 退出聊天框的消息列表，进入聊天框的输入框
	alt+l 退出聊天框的消息列表，进入联系人/群聊选择侧边栏
	esc   退出聊天框的消息列表，主窗口最小化
}
alt+s 进入聊天框的输入框
{
    enter 发送聊天框的输入框中的消息
	
	alt+m 退出聊天框的输入框，进入聊天框的消息列表
	alt+l 退出聊天框的输入框，进入联系人/群聊选择侧边栏
	esc   退出聊天框的输入框，主窗口最小化
}
alt+l 进入联系人/群聊选择侧边栏
{
    ↑     切换到上一个联系人
	↓     切换到下一个联系人
	
	enter 退出联系人/群聊选择侧边栏，进入当前选中的联系人/群聊的聊天框的输入框（独立窗口，按esc键退出该窗口）
	alt+m 退出联系人/群聊选择侧边栏，进入聊天框的消息列表
	alt+s 退出联系人/群聊选择侧边栏，
	esc   退出联系人/群聊选择侧边栏，主窗口最小化
}
