# Emacs 快速指南 · 功能参考手册

> 本手册基于 GNU Emacs 官方教程（TUTORIAL.cn）整理，按功能分类编排，供用户快速查询。
>
> **命令约定**：
> - **C-<chr>**：按住 `Ctrl` 键再按字符。
> - **M-<chr>**：按住 `Alt` / `Option` 键再按字符。若无此键，可先按 **`<ESC>`** 再按字符（注意顺序）。
> - **<DEL>**：退格键（Backspace）；**<Return>**：回车键（Enter）；**<SPC>**：空格键。


## ⚡ 急救命令速查卡（必记）
| 场景 | 命令 |
|------|------|
| **打开文件** | `C-x C-f` |
| **保存文件** | `C-x C-s` |
| **退出 Emacs** | `C-x C-c` |
| **取消当前命令 / 救命** | `C-g` |
| **撤销修改** | `C-/` |
| **搜索** | `C-s`（向前） / `C-r`（向后） |


## 一、启动、退出与中断
| 命令 | 功能 |
|------|------|
| `C-x C-c` | **退出 Emacs**（若有未保存修改会提示） |
| `C-z` | **挂起 Emacs**（仅限终端，回到 Shell，用 `fg` 恢复；图形界面下通常最小化） |
| `C-g` | **取消/终止**当前正在执行的命令、取消数字参数、退出小缓冲（Minibuffer） |
| `ESC ESC ESC` | **通用“暴力退出”**：退出递归编辑、关闭多余窗格或退出小缓冲 |


## 二、光标移动（导航）

### 2.1 按字符移动
| 命令 | 功能 |
|------|------|
| `C-f` | 向前（forward）移动 **一个字符** |
| `C-b` | 向后（backward）移动 **一个字符** |

### 2.2 按词移动（中文环境下为按空格/标点）
| 命令 | 功能 |
|------|------|
| `M-f` | 向前移动 **一个词** |
| `M-b` | 向后移动 **一个词** |

### 2.3 按行移动
| 命令 | 功能 |
|------|------|
| `C-n` | 移动到 **下一行**（next） |
| `C-p` | 移动到 **上一行**（previous） |
| `C-a` | 移动到 **行首** |
| `C-e` | 移动到 **行尾** |

### 2.4 按句移动
| 命令 | 功能 |
|------|------|
| `M-a` | 移动到 **句首** |
| `M-e` | 移动到 **句尾** |

### 2.5 按屏/页/全局移动
| 命令 | 功能 |
|------|------|
| `C-v` | 向下滚动 **一屏** |
| `M-v` | 向上滚动 **一屏** |
| `C-l` | **重绘屏幕**：将光标所在行移到**中央** → 再按移到**顶端** → 再按移到**底端** |
| `M-<` | 移动到 **整个文件的开头**（注意 `<>` 需按 Shift） |
| `M->` | 移动到 **整个文件的末尾** |

> **💡 数字参数（重复次数）**：大部分移动命令可加前缀参数。
> - 用法：先按 `C-u <数字>`，再按命令。如 `C-u 8 C-f` 向前移动 8 个字符。
> - 特例：给 `C-v` / `M-v` 加参数时，滚动的是**行数**而非屏数。如 `C-u 8 C-v` 向下滚动 8 行。


## 三、编辑操作

### 3.1 插入文本
| 操作 | 功能 |
|------|------|
| 直接键入任意字符 | 在当前光标处插入该字符 |
| `<Return>` | 插入换行符（并根据上下文**自动缩进**） |
| `C-u <数字> <字符>` | 重复插入同一字符 N 次（如 `C-u 10 *` 插入 10 个星号） |

### 3.2 删除（Delete）与移除（Kill）—— 概念要分清
> **删除（Delete）**：内容被丢弃，**不可**通过 `C-y` 召回（但可撤销）。
> **移除（Kill）**：内容被暂存，**可以**通过 `C-y` 召回（类似“剪切”）。

| 命令 | 功能 | 性质 |
|------|------|:---:|
| `<DEL>` | 删除光标**前**一个字符 | ❌ 删除 |
| `C-d` | 删除光标**后**一个字符 | ❌ 删除 |
| `M-<DEL>` | **移除**光标前一个词 | ✅ 移除 |
| `M-d` | **移除**光标后一个词 | ✅ 移除 |
| `C-k` | **移除**从光标到**行尾**的字符（不含换行符） | ✅ 移除 |
| `M-k` | **移除**从光标到**句尾**的字符 | ✅ 移除 |
| `C-w` | **移除**从标记（Mark）到光标之间的**区域** | ✅ 移除 |
| `M-w` | **复制**从标记到光标之间的区域（原文保留） | ✅ 复制 |

> **⚠️ 重要特例（关于 `C-d` 和 `<DEL>`）**：
> 当给 `C-d` 或 `<DEL>` **加上前缀参数**（如 `C-u 10 C-d`）时，它们的行为会从“删除”**自动转变为“移除”**，此时被删的内容可以被 `C-y` 召回。

### 3.3 设置标记（Mark）与操作区域
| 命令 | 功能 |
|------|------|
| `C-<SPC>` 或 `C-@` | 在当前光标位置**设置标记**（Mark） |
| 移动光标到另一端 | 标记与光标之间的文字会**高亮**显示（区域） |
| `C-w` | **移除（剪切）** 该区域内的文字 |
| `M-w` | **复制** 该区域内的文字（不删除原文） |
| `C-y` | 在光标处**召回（粘贴）** 最近移除的内容 |

> **💡 解决 `C-<SPC>` 被输入法占用**：
> 在中国大陆的 Windows / Linux 上，`C-<SPC>` 通常被中英文输入法拦截。Emacs 提供了备用键：**`C-@`**（即 `Ctrl + Shift + 2`）可完全替代 `C-<SPC>` 设置标记。

### 3.4 召回（Yank）与轮换
| 命令 | 功能 |
|------|------|
| `C-y` | 在光标处插入最近一次移除的内容（召回/粘贴） |
| `M-y` | **仅在 `C-y` 之后使用**：轮换到前一次移除的内容（可连续按多次循环） |

### 3.5 撤销（Undo）
| 命令 | 功能 |
|------|------|
| `C-/` | 撤销上一次命令造成的修改（连续按撤销更早的操作） |
| `C-_` | 同上（部分终端可用 `C--`，即 Ctrl + 减号） |
| `C-x u` | 同上（功能相同，按键稍长） |


## 四、文件操作

### 4.1 基本文件命令
| 命令 | 功能 |
|------|------|
| `C-x C-f` | **打开/创建文件**（在小缓冲中输入文件名；不存在则创建空文件） |
| `C-x C-s` | **保存当前文件**（首次保存会生成备份文件 `原文件名~`） |
| `C-x s` | **保存多个缓冲区**：逐个询问是否保存已修改的缓冲区 |

### 4.2 自动保存与恢复
| 功能 | 说明 |
|------|------|
| **自动保存文件** | 文件名格式为 `#原文件名#`（如 `#hello.c#`），在正常存盘后自动删除 |
| `M-x recover-file` | **从自动保存文件恢复数据**：打开原文件后执行，按提示确认 `yes` 即可 |


## 五、缓冲区（Buffer）管理
| 命令 | 功能 |
|------|------|
| `C-x C-b` | 列出当前所有的缓冲区（`*Buffer List*`） |
| `C-x b <名称>` | 切换到指定名称的缓冲区（输入名称时可用 `<Tab>` 补全） |
| `C-x s` | 保存所有已修改且未存盘的缓冲区（逐个确认） |
| 特殊缓冲区 | `*Messages*`（消息日志）、`*scratch*`（草稿）等无对应文件，属于临时缓冲区 |


## 六、窗格（Window）与窗口（Frame）管理

> **术语区分**：在 Emacs 中，**窗格（Window）** 是同一个窗口（Frame）内分割出的编辑区域；**窗口（Frame）** 是操作系统层面的独立顶层窗口。

### 6.1 窗格操作（分屏）
| 命令 | 功能 |
|------|------|
| `C-x 1` | 只保留当前窗格，**关闭其他所有窗格** |
| `C-x 2` | 将当前窗格**垂直拆分为上下两个窗格** |
| `C-x o` | 将光标**切换到下一个窗格**（循环遍历） |
| `C-M-v` | **滚动另一个窗格**的内容（当前光标所在窗格不动） |
| `C-x 4 C-f` | 在新窗格中打开文件（若当前仅一个窗格，则自动水平拆分） |

> **💡 关于 `C-M-v` 的按键**：
> - 有 META 键：同时按住 `Ctrl` + `Alt` + `v`。
> - 无 META 键：先按 `<ESC>`，再按 `C-v`（注意顺序，`C-<ESC>-v` 无效）。

### 6.2 窗口（Frame）操作（图形界面）
| 命令 | 功能 |
|------|------|
| `C-x 5 2` | 创建一个**新的独立顶层窗口**（Frame） |
| `C-x 5 0` | 关闭当前选中的顶层窗口 |

### 6.3 鼠标/图形界面交互（GUI）
| 操作 | 功能 |
|------|------|
| 鼠标拖动**滚动条** | 滚动当前窗格的内容 |
| **鼠标滚轮** | 上下滚动当前窗格 |
| 鼠标点击其他窗格 | 将光标焦点切换到该窗格（等效于 `C-x o`） |
| 点击窗口的“X”按钮 | 关闭该顶层窗口（若为最后一个则退出 Emacs） |


## 七、渐进式搜索
| 命令 | 功能 |
|------|------|
| `C-s` | **向前（向下）** 渐进搜索 |
| `C-r` | **向后（向上）** 渐进搜索 |
| **搜索中**按 `C-s` | 跳到**下一个**匹配位置 |
| **搜索中**按 `C-r` | 跳到**上一个**匹配位置 |
| **搜索中**按 `<DEL>` | **撤回**到上一次匹配位置（若已无匹配，则删除搜索词末字符） |
| **搜索中**按 `<Return>` | 结束搜索，光标**停留在**当前位置 |
| **搜索中**按 `C-g` | **取消搜索**，光标**回到搜索起始位置** |

> **💡 搜索时的回退技巧**：
> 若输入了 “cursor”，按 `<DEL>` 删掉 “r”，光标会立刻跳回 “curso” 的命中位置，而不是原地不动。这能在拼写错误时帮你直观回退。


## 八、主模式与辅模式
### 8.1 主模式（Major Mode）
| 命令 | 功能 |
|------|------|
| `M-x fundamental-mode` | 切换到 **Fundamental 模式**（默认，无特殊语法） |
| `M-x text-mode` | 切换到 **Text 模式**（编辑自然语言，标点符号处理更智能） |
| `C-h m` | 查看当前主模式的详细文档，并**列出所有激活的辅模式** |

### 8.2 辅模式（Minor Mode）
| 命令 | 功能 |
|------|------|
| `M-x auto-fill-mode` | **开关自动折行**模式（输入超过行宽时自动换行） |
| `C-x f <数字>` | 设置行宽边界（如 `C-u 20 C-x f` 设为 20 字符） |
| `M-q` | 手动对**当前段落**重新折行（需在 Auto Fill 模式下生效） |


## 九、帮助系统
| 命令 | 功能 |
|------|------|
| `C-h ?` | 列出所有可用的帮助选项 |
| `C-h c <组合键>` | 显示该组合键对应的**命令名**（简要说明） |
| `C-h k <组合键>` | 在新窗格显示该组合键对应的**完整函数文档** |
| `C-h x <命令名>` | 显示指定**命令名的文档**（如 `C-h x previous-line`） |
| `C-h v <变量名>` | 显示 Emacs 变量的文档（用于定制行为） |
| `C-h a <关键词>` | **相关命令搜索**：列出所有命令名中包含该关键词的命令 |
| `C-h m` | 显示当前**主模式**的文档以及所有**激活的辅模式**列表 |
| `C-h i` | 打开 **Info 阅读器**（内置手册系统，按 `?` 查看使用指南） |
| `C-h r` | 打开 **Emacs 使用手册**（从 Help 菜单也可进入） |

> **💡 若 `C-h` 不生效**：某些终端把 `C-h` 映射为删除键，可尝试 **F1** 键或 `M-x help <Return>`。


## 十、扩展命令（Extended Commands）
| 命令/前缀 | 功能 |
|------|------|
| `C-x <字符>` | `C-x` 开头的扩展命令（如文件、窗格操作） |
| `M-x <命令名>` | 按**命令名**执行扩展命令（输入时可用 `<Tab>` 自动补全） |

### 常用扩展命令示例
| 命令 | 功能 |
|------|------|
| `M-x replace-string` | **全局替换**：输入被替换词 → 输入新词（光标后生效） |
| `M-x recover-file` | 从自动保存文件恢复数据 |
| `M-x dired` | 进入**目录编辑器**（文件管理），可对文件进行移动/复制/删除/重命名 |
| `M-x list-packages` | 打开**包管理界面**（安装/卸载/浏览扩展） |
| `M-x auto-fill-mode` | 开关自动折行 |
| `M-x text-mode` | 切换到文本模式 |


## 十一、高级与杂项功能
### 11.1 被禁用的命令（Disabled Commands）
为防止新手误操作，部分危险命令（如 `C-x C-l`）默认被禁用。
- **触发时行为**：Emacs 会弹出提示，询问是否要继续执行。
- **选择**：按 **空格** 执行本次，按 **n** 取消。
- **永久启用**：后续可通过 `M-x enable-command` 永久开启该命令。

### 11.2 递归编辑（Recursive Editing）
- **识别**：当状态栏显示 **`[(Fundamental)]`**（比平常多一层方括号）时，表示处于递归编辑中。
- **退出**：**`ESC ESC ESC`** 可逐层暴力退出。
- **注意**：`C-g` 只能取消当前层级的命令，**无法**退出递归编辑本身。

### 11.3 回显区（Echo Area）提示
- 当 Emacs 检测到你输入多字符命令（如 `C-x 4`）后停顿过久，**回显区会自动弹出后续可用的按键提示**，这是 Emacs 的防呆机制，不是系统卡顿。

## 附录 B：高频 Tips 汇总

1. **`C-@` 替代 `C-<SPC>`**：当设置标记（Mark）被输入法拦截时，用 `Ctrl + Shift + 2`。
2. **`C-g` 能救命**：任何卡死、输错命令、不想继续时，狂按 `C-g` 都能安全中止。
3. **用 `<Tab>` 偷懒**：在 `M-x` 后、`C-x b` 后、`C-x C-f` 后，都可以按 `<Tab>` 自动补全。
4. **`C-k` 的数字参数**：`C-u 2 C-k` 移除两整行（含换行）；`C-u 1 C-k` 仅移除本行内容（不含换行），与普通 `C-k` 不同。
5. **查看当前所有快捷键**：`C-h b`（虽未在教程主文，但极其常用，笔者补充）。
6. **撤销（Undo）多按几次**：可以撤销更早的历史修改，不仅限于上一步。
7. **`M-y` 必须紧跟在 `C-y` 之后**：想翻找更早的移除内容，必须先 `C-y`，再 `M-y`。
