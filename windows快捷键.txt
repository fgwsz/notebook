>> 打开终端
   win+r 打开输入框，输入powershell，输入回车键就可以打开终端

>> 打开搜索框
   win+s 打开搜索框窗口，光标进入搜索框
         输入后点击回车，进入选中项窗口
   esc   退出窗口
   
>> 切换任务栏
   按住alt键，按数次tab键来选择任务窗口，松开alt键跳转到选择的任务窗口
   按住alt键，使用方向键也可以选择窗口，松开alt键跳转到选择的任务窗口

>> 运行程序
   把常用的软件的主程序的父路径设置成环境变量。
   这样我们可以直接调用{exe_file_name} 来调用环境变量中文件夹下的一些可执行程序。
   QQ       # 启动QQ
   WeChat   # 启动微信
   msedge   # 启动Edge浏览器
   explorer # 启动资源管理器
   也有一些在脚本中使用的土办法：
   Start-Process {exe_path} # powershell命令调用方式
   start {exe_path}         # cmd命令调用方式
 
>> 窗口操作
   win+↑  最大化窗口
   alt+f4 关闭窗口
   win+d  显示桌面/恢复到原先的显示界面

>> 一些通用的快捷键

alt 按一下然后进入当前窗口的菜单栏选项
   +space 切换
   +tab   切换
	+alt   退出alt模式
	
ctrl+s 保存
ctrl+c 复制
ctrl+v 粘贴
ctrl+z 撤销
ctrl+y 取消撤销
ctrl+d 删除文件/复制光标所在的一行内容，新建下一行，粘贴内容到新建的行中
ctrl+x 剪切
F2     重命名
win+l  锁屏
win+x  打开菜单栏选项卡
win+space 切换输入法

>> 用键盘选中文本
shfit + ↑ 向上选择文本  
shfit + ↓ 向下选择文本  
shfit + ← 向左选择文本
shfit + → 向右选择文本

>> windows 键失效的恢复方式
Fn + windows 键可以恢复

>> explorer.exe 的快捷键
使用alt 切换到最上面一栏 默认选中`新建`按钮
  之后可以使用← →键来切换最上面的一栏按钮，
  点击回车可以按下这些按钮
使用tab键可以将选中项目切换到下面的一些按钮
  当选中文件的时候按回车是打开文件
  当选中文件夹的时候按回车是打开文件夹
alt+↑  切换到上一级目录
alt+←  回退到上一次打开的目录
alt+→  取消回退，回到原先打开的目录

>> word 快捷键(wps通用)
ctrl+b     字体加粗
ctrl+alt+v 粘贴(粘贴之前可以选择粘贴的格式)

>> wps 快捷键
ctrl+tab  切换标签页
ctrl+w    关闭标签页

>> vscode 快捷键
ctrl+tab  切换标签页

>> opera 浏览器
alt 打开菜单栏选项卡

>> 网页视频播放器
← 快退5秒
→ 快进5秒
space 播放/暂停播放
