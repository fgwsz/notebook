>> 打开终端
   <win-r> 打开输入框，输入powershell，输入回车键就可以打开终端

>> 打开搜索框
   <win-s> 打开搜索框窗口，光标进入搜索框
         输入后点击回车，进入选中项窗口
   <esc>   退出窗口
   
>> 切换任务栏
   按住<alt>键，按数次<tab>/<shift-tab>键来选择任务窗口，松开<alt>键跳转到选择的任务窗口
   按住<alt>键，使用方向键也可以选择窗口，松开<alt>键跳转到选择的任务窗口

>> 窗口操作
   <win-↑>  最大化窗口
   <alt-F4> 关闭窗口
   <win-d>  显示桌面/恢复到原先的显示界面

>> 一些通用的快捷键
<alt> 按一下然后进入当前窗口的菜单栏选项
    <tab>         切换到下一个
    <shift+tab>   切换到上一个

<ctrl-a> 全选
<ctrl-s> 保存
<ctrl-c> 复制
<ctrl-v> 粘贴
<ctrl-z> 撤销
<ctrl-y> 取消撤销
<ctrl-d> 删除文件/复制光标所在的一行内容，新建下一行，粘贴内容到新建的行中
<ctrl-x> 剪切
F2       重命名
<win-l>  锁屏
<win-x>  打开菜单栏选项卡
<win-space> 切换输入法
<win-e>  打开文件浏览器
<space>  选中/取消选中按钮

>> 用键盘选中文本
<shfit-↑> 向上选择文本
<shfit-↓> 向下选择文本
<shfit-←> 向左选择文本
<shfit-→> 向右选择文本

>> windows 键失效的恢复方式
<Fn-win> 键可以恢复

>> explorer.exe 的快捷键
使用<alt>切换到最上面一栏 默认选中`新建`按钮
  之后可以使用← →键来切换最上面的一栏按钮，
  点击回车可以按下这些按钮
使用<tab>/<shift+tab>键可以将选中项目切换到下面的一些按钮
    <tab>         切换到下一个
    <shift+tab>   切换到上一个
    <enter>
        选中按钮时点击按钮
        选中文件时打开文件
        选中文件夹时打开文件夹
<alt-↑>  切换到上一级目录
<alt-←>  回退到上一次打开的目录
<alt-→>  取消回退，回到原先打开的目录
<alt-d>  进入地址输入框
    可以在地址输入框输入powershell并回车来在当前目录下打开powershell
    可以在地址输入框输入一个文件路径并回车来跳转到该文件路径
    <esc> 退出地址输入框
<alt-f>  打开右键菜单
            ↑ 切换到上一个待选项
            ↓ 切换到下一个待选项
            <enter> 选中待选项/打开子菜单
            <esc> 退出右键菜单
<ctrl-N> 新建文件夹

>> word 快捷键(wps通用)
<ctrl-b>     字体加粗
<ctrl-alt-v> 粘贴(粘贴之前可以选择粘贴的格式)

>> wps 快捷键
<ctrl-tab>          切换到下一个标签页
<ctrl-shift-tab>    切换到上一个标签页
<ctrl-w>            关闭当前标签页

>> vscode 快捷键
<ctrl-tab>          切换到下一个标签页
<ctrl-shift-tab>    切换到上一个标签页

>> opera 浏览器
<alt> 打开菜单栏选项卡

>> 网页视频播放器
←       快退5秒
→       快进5秒
<space> 播放/暂停播放
↑       增大10%音量
↓       减少10%音量

>> edge 浏览器(opera浏览器通用)
<alt>            光标放置到...按钮上，按回车就打开菜单选项卡
<ctrl-d>         收藏当前标签页
<ctrl-w>         关闭当前标签页
<ctrl-tab>       切换到下一个标签页
<ctrl-shift-tab> 切换到上一个标签页
<ctrl-j>         打开下载标签栏小窗口，之后可以使用tab键来选择选项按钮，<enter>来点击按钮，<esc>键退出小窗口
<ctrl-p>         打印当前网页
<alt-d>          进入标签页地址栏
<alt-←>          后退
<alt-→>          前进
<alt-p>          固定/取消固定标签页
<space>          向下滚动
<shift-space>    向上滚动

>> powershell 快捷键
<ctrl-,>    弹出设置窗口
